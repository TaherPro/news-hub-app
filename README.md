# News Hub App

A simple and functional News Hub web application built with React, HTML/CSS, and JavaScript, using the NewsAPI to fetch real-time news articles.

Main Features
* Browse top US news headlines
* Click any article to read the full story
* Filter news by categories (General, Business, Sports, Technology, Health)

Tech Stack
* React 
* Context for state management
* NewsAPI for fetching articles
* HTML + CSS for layout and styling
